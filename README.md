# Política de Privacidad – SwMusic

SwMusic es una aplicación en fase de pruebas que permite a los usuarios acceder mediante Google Sign-In para fines de autenticación.

## Información que recopilamos
Cuando el usuario inicia sesión con Google, podemos recopilar:
- Dirección de correo electrónico
- Nombre visible del perfil
- Identificador único de usuario

## Uso de la información
La información recopilada se utiliza exclusivamente para:
- Permitir el acceso y autenticación en la aplicación
- Probar y mejorar el funcionamiento del app durante la fase de pruebas

No vendemos, alquilamos ni compartimos información personal con terceros.

## Servicios de terceros
SwMusic utiliza Google Sign-In y Google Play Services, los cuales pueden recopilar información conforme a sus propias políticas de privacidad.

## Seguridad
Aplicamos medidas razonables para proteger la información de los usuarios.

## Cambios
Esta política puede actualizarse conforme avance el desarrollo del app. Cualquier cambio será publicado en esta misma página.
